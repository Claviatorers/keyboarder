Руководство пользователя

Назначение игровой системы

Система клавиатурный тренажёр «KEYBOARD GYM» предназначена для обучения пользователя слепой десятипальцевой печати на клавиатуре с ведением его статистики тренировок.

Авторизация в системе

После запуска системы откроется окно автоматизации. Пользователь должен ввести логин и нажать «Вход». Если вход в программу от пользователя осуществляется впервые, то необходимо пройти процедуру регистрации, нажав кнопку «Регистрация».

Для завершения регистрации необходимо нажать «Зарегистрироваться». В случае если пользователь с таким логином присутствует в системе, система выведет сообщение об ошибке.

После успешного прохождения авторизации открывается главная форма обучаемого или администратора.

Администрирование

Если пользователь – администратор, то перед ним откроется экранная форма «Администрирование», где будет отображено главное меню возможностей администратора. Администратор может управлять учётными записями пользователей, заняться настройкой тренажёра, а также работать с упражнениями.

Настройка тренажера

Для того чтобы настроить тренажёр, нужно в главном окне «Администрирование» нажать кнопку «Настройка тренажёра». Появится окно «Настройка тренажёра», в котором можно будет для выбранного уровня задать время нажатия одного символа, максимальную длину упражнения, процент допустимых ошибок и допустимые символы. В системепредусмотрено 3 уровня сложности, каждый из которых администратор может изменить. Для сохранения введенных данных нужно нажать кнопку «Сохранить».

Просмотр учетных записей пользователей

Чтобы просмотреть информацию о пользователях администратору нужно в главном окне «Администрирование» нажать на кнопку «Учетные записи пользователей». В открывшемся окне «Учетные записи пользователей» будет отображаться информация о пользователях системы. Администратор может просмотреть информацию по отдельному пользователю, путем выбора из левого меню пользователя. Или может просмотреть статистику по всем пользователям системы, путем нажатия на кнопку «Общая статистика».

Настройка упражнений

Для того чтобы начать настройку упражнений, нужно на главной форме «Администрирование» нажать кнопку «Настройка упражнений». Откроется новое окно «Настройка упражнений», в котором будут отображаться все основные функции при работе с упражнениями: добавить, редактировать, удалить.

Добавление или изменение упражнения

Для составления нового упражнения нужно в форме «Работа с упражнениями» выбрать уровень сложности и нажать кнопку «Добавить». Откроется новое окно. Далее нужно ввести новое упражнение вручную или же, путем нажатия кнопки автогенерация, сгенерировать упражнение автоматически.

Для добавления или именения введенного упражнения в систему необходимо нажать кнопку «Сохранить».

Удаление упражнения

Для удаления упражнения администратору необходимо в окне «Настройка упражнений» выбрать уровень сложности, выбрать упражнение из списка и нажать кнопку «Удалить». Упражнение удалиться из базы данных.

Обучение

Если пользователь – обучаемый, то перед ним откроется экранная форма «Обучение», где будут отображены последние результаты тренировки. Обучаемый может начать новую тренировку, посмотреть статистику, а также вызвать справку.

Редактирование учётной записи

Чтобы изменить учётную запись необходимо в главном окне «Обучение» нажать на кнопку редактирования возле учетной записи, в этом случае всплывёт окно «Редактирование учётной записи», где можно изменить данные. Для сохранения изменений нужно нажать «Сохранить».

Настройка тренировкий

Для начала настройки тренировки нужно выбрать уровень сложности в окне справа и выбрать из списка доступное упражнение.

Для получения справки о распределении нагрузки на уровнях и получения информации об остальных доступных функциях необходимо нажать «Справка».

После запроса на прохождение тренировки откроется окно «Тренировка». Пользователю нужно установить руки в правильное положение на нужные клавиши.

Просмотр статистики

Чтобы просмотреть статистику своих тренировок, обучаемому необходимо в главном окне «Обучение» нажать на кнопку «Статистика», в этом случае откроется окно «Статистика пользователя», в котором будут отображаться результаты пройденных тренировок.
